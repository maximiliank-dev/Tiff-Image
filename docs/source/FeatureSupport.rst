=============================
Tiff Feature Support
=============================

The library supports the following TIFF Features:

.. table::
   :align: left

   +----------------------+-----------+
   | Image Type           | Supported |
   +======================+===========+
   | Bitlevel Images      | ✅        |
   +----------------------+-----------+
   | Grayscale Images     | ✅        |
   +----------------------+-----------+
   | RGB Images           | ✅        |
   +----------------------+-----------+


.. table::
   :align: left

   +----------------------+-----------+
   | Compression          | Supported |
   +======================+===========+
   | PackBits Compression | ✅        |
   +----------------------+-----------+
   | LZW Compression      | 🚧        |
   +----------------------+-----------+
